OpenSMTPD Документация (man-страницы)

Web руководства доступны в самой OpenBSD. Эти руководства относятся к последней версии OpenSMTPD.

  • smtpd — демон SMTP (Simple Mail Transfer Protocol)
  • smtpd.conf — конфигурационный файл SMTP демона
  • smtpctl — менеджер SMTP демона
  • newaliases — alias mappings генератор
  • makemap — mappings генератор
  • aliases — описание формата aliases файлов
  • forward — формат forwarding файлов
  • table — описание формата таблиц smtpd
  • sendmail — просмотр очереди писем (a mail enqueuer) для smtpd

Протокол SMTP, реализованный в OpenSMTPD, описан в RFC 5321.
