OpenSSH Почтовые рассылки

Рассылка для разработчиков (private)

Связанные с безопасностью вопросы могут быть направлены в закрытую от посторонних глаз рассылку openssh@openssh.com, которая предназначена только для OpenSSH разработчиков. Она не подходит для обсуждения общих вопросов.

Новостная рассылка

Для Linux/Unix порта есть своя почтовая рассылка openssh-unix-announce@mindrot.org. Эта рассылка доступна только для чтения, трафик тут очень низкий. Предназначена в первую очередь для анонсов.

Информация о настройках и о том, как подписатья, доступна тут.

Рассылка для разработчиков (public)

Почтовая рассылка openssh-unix-dev (openssh-unix-dev@mindrot.org) посвящена разработке обоих версий OpenSSH.

Трафик рассылки от нуля до двадцать писем в день. Основня (единственная разжешенная) тема - хакинг обоих версий OpenSSH (портированная и для OpenBSD), работа над документацией и обсуждение и/или исправление багов (если вы читали руководства). НЕ РАЗРЕШАЕТСЯ: HTML, перегруженный форматированием текст и большие по размеру attachments. Разрешается прикреплять только небольшие (например diff) файлы. Ваши просьбы о помощи, которые демонстрируют недостаточное знакомство с OpenSSH может привести только к "RTFM"-ответам, поэтому, пожалуйста, прежде чем что-либо спросить, ознакомтесь с документацией.

Обсуждения коммерческих продуктов в этой рассылке являются *оффтопом*, за исключением тех случав, когда речь идет о разработке OpenSSH (обсуждение взаимодействия).

Список открыт для всех, хотя non-subscribers могут быть целью для одноразовых проверок адреса. Архив доступен на MARC.

Более подробная информация об этой рассылке находится на странице openssh-unix-dev@mindrot.org.
