OpenBGPD Документация

Web руководства доступны в самой OpenBSD. Эти руководства относятся к последней версии OpenBGPD.

  • bgpd(8) — BGP (Border Gateway Protocol) демон
  • bgpd.conf(5) — конфигурационный файл демона BGP (Border Gateway Protocol)
  • bgpctl(8) — клиент для управления BGP (Border Gateway Protocol) демоном

Протокол BGP, реализованный в OpenBGPD, описан в RFC 4271.
